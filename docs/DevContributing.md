## Contributing

Contributions are welcome. If you want to improve the project, please follow this process:

1. **Fork the repository** to your GitHub account.
2. **Clone your fork locally** with a git client.
   ```sh
   git clone ./
   ```
3. **Create a dedicated branch** for your changes.
   ```sh
   git checkout -b new-feature-x
   ```
4. **Implement and test** your changes locally.
5. **Commit with a clear message** that describes the update.
   ```sh
   git commit -m 'Implemented new feature x.'
   ```
6. **Push the branch** to your remote fork.
   ```sh
   git push origin new-feature-x
   ```
7. **Open a pull request** against the main repository and explain the motivation behind the change.
8. **Review and iterate** until the contribution is ready to merge.
